# SmartBuy-Backend

Backend oficial do Sistema Smart Buy BH e Ordens de Serviço.

## Responsabilidades

- APIs e contratos HTTP;
- autenticação e autorização;
- regras de negócio;
- acesso ao PostgreSQL/Supabase;
- migrations e políticas RLS;
- integrações externas;
- auditoria e processamento seguro.

## Governança

- código integralmente nativo;
- código legado não pode ser incorporado;
- arquivos manuais limitados a 300 linhas;
- God Objects e God Modules são proibidos;
- regras de negócio não podem ser executadas pelo frontend;
- alterações devem ocorrer por Pull Request e testes.
